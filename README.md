# ECUST 自动控制原理 实验报告
## 任务说明（任务B）
1.	利用提供的temperature.csv，使用经典辨识方法（两点法等）辨识出加热炉对象的输入输出模型，并验证辨识方法的准确性。
2.	根据辨识出的模型，设计 PID 控制器对系统进行闭环控制，并使用智能优化算法调整PID控制器参数，使加热炉温度稳定控制在35℃。同时，计算并分析动态指标和稳态指标。
3.	提交的材料需包含完整的问题分析、算法设计过程以及最终的控制效果。
